# ConventCommons
Convent Commons for Mendix

Module content:
. DataGrid2, auto select (first) row
. EnumReflection, opens full potential of enums

Shorthand implementation guide.
1. Import module ConventCommons.mpk in your app.
2. DataGrid2 can be used out of the box.
3. To use EnumReflection add microflow ASU_EnumReflection to your start-up microflow.
4. Optionally add page Enumerations_Overview to your navigation to have an insight in reflection data.
5. Your good to go to use the micro- and nanoflows and the Java- and JavaScriptActions in the folders under Enumerations.
